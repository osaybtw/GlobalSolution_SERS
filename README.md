Este projeto foi desenvolvido para a disciplina de Soluções em Energias Renováveis e Sustentáveis. O objetivo é simular o monitoramento de uma missão espacial experimental por meio da análise de dados energéticos e operacionais informados pelo usuário.

O sistema recebe informações sobre temperatura, nível de energia da bateria, geração solar, consumo energético, qualidade da comunicação e status do módulo principal. A partir desses dados, é realizada uma análise das condições atuais da missão, identificando possíveis situações de risco e apresentando alertas automáticos.

Além da geração de alertas, o programa executa ações simples de forma automática quando detecta condições críticas. Caso a energia esteja baixa ou o consumo seja maior que a geração solar, o sistema ativa um modo de economia de energia e reduz o consumo. Em situações de superaquecimento, é acionado um sistema de resfriamento para reduzir a temperatura. Quando a energia atinge níveis muito baixos, módulos secundários podem ser desligados para preservar recursos essenciais da missão.

A utilização da geração solar representa a aplicação de uma fonte de energia renovável no contexto da missão. A comparação entre geração e consumo permite avaliar a sustentabilidade da operação, indicando se a energia produzida é suficiente para manter os sistemas em funcionamento.

Ao final da execução, o sistema apresenta um relatório contendo os dados informados, os alertas identificados, as ações executadas automaticamente e a situação geral da missão após as correções aplicadas.
